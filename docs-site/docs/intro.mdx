# AOH — Agentic Ops Harness

AOH turns runbooks into portable, evaluated, git-versioned skills — "Superpowers for
Ops" — so cheap or local models can execute them repeatably while frontier models stay
reserved for authoring skills and unblocking the hard cases. One pack format compiles to
many agent runtimes (Hermes today; Claude Code, Codex, and Goose next), the way one
Ansible role runs against many inventories. AOH organizes, packages, validates, and
adapts a pack — it never executes; the runtime you point it at does.
